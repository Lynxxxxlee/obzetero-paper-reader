# Paper Note Template

Every generated note must keep these sections.

```markdown
---
title: "Paper title"
zotero_key: KEY
zotero_link: "zotero://select/library/items/KEY"
item_type: "preprint"
status: unread
date: "2026-01-01"
doi: ""
url: ""
authors:
  - "Author"
tags:
  - zotero
  - paper
---

# Paper title

## AUTO: Metadata

## AUTO: Chinese Reading Notes

### 一句话结论
### 核心贡献
### 方法拆解
### 技术细节
### 实验设置
### 结果解读
### 局限性
### 可复现性
### 可借鉴点

## SYNC: Zotero Note

## MANUAL: My Notes

## MANUAL: Questions

## MANUAL: Next Actions

## Abstract
```

Only rewrite `AUTO:` sections by default. Preserve `MANUAL:` sections.
